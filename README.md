# PROJETO-REACTNATIVE---LIPE

## Currículo de Phelipe — resumo para este repositório

Este repositório contém o currículo de Phelipe em formato versionado. O objetivo deste README é explicar o que está no currículo, como navegar pelas seções e como atualizar o conteúdo.

### Principais seções do currículo
- Resumo profissional: visão geral da carreira e objetivo profissional.
- Experiência: posições anteriores, responsabilidades principais e resultados relevantes.
- Habilidades técnicas: foco em React Native, JavaScript/TypeScript, mobile development, bibliotecas e ferramentas relacionadas.
- Projetos: descrições de apps ou módulos desenvolvidos (com links para o código quando disponíveis).
- Educação e certificações: formação acadêmica e cursos relevantes.
- Idiomas e contato: níveis de proficiência e formas de contato (e‑mail, LinkedIn).

### Destaques esperados (exemplo)
- Desenvolvimento de aplicações móveis com React Native (iOS/Android).
- Integração com APIs REST/GraphQL, gerenciamento de estado (Redux / Context), navegação e testes.
- Entrega de features, correção de bugs e otimização de performance em projetos reais.
- Experiência com CI/CD, testes automatizados e publicação em lojas.
